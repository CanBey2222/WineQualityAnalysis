# Wine Quality Analysis

This project presents an Exploratory Data Analysis (EDA) study on a wine quality dataset. The main goal is to examine the relationships between the chemical properties of wine and the `quality` variable, identify meaningful patterns, and generate insights through statistical summaries and visual analysis.

## Project Objective

This study focuses on the following questions:

- Which variables may have an impact on wine quality?
- Are there any strong correlations between the features?
- How do feature distributions change across different quality levels?
- What meaningful insights can be extracted from the dataset through visualization and descriptive analysis?

## Dataset

The dataset used in this project is stored in the `13-WineQT.csv` file.

### Dataset Summary

- Number of observations: `1143`
- Number of features: `13`
- Target variable: `quality`

### Features

The main columns in the dataset are:

- `fixed acidity`
- `volatile acidity`
- `citric acid`
- `residual sugar`
- `chlorides`
- `free sulfur dioxide`
- `total sulfur dioxide`
- `density`
- `pH`
- `sulphates`
- `alcohol`
- `quality`
- `Id`
